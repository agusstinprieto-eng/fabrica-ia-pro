AI ANALYSIS METHODOLOGY

How Gemini Analyzes Operations with Only 6 Frames

Video Processing Limits

  • > Maximum Duration: 2 minutes
  • > Frames Extracted: 6 frames (distributed uniformly)
  • > Image Quality: JPEG 0.8 compression
  • > Token Optimization: ~85% reduction vs. full video analysis

Analysis Pipeline

  1. 01
    Sequence Recognition: The 6 frames capture the complete operation cycle (start → intermediate steps → finish). AI identifies the operation type and phase transitions.
  2. 02
    Movement Detection: Computer vision tracks hand/body position changes between frames to calculate distances, angles, and complexity (finger count, rotation, precision required).
  3. 03
    Standard Time Database Matching: Detected movements are matched against industrial time standards:
    • • Motion Analysis: "Reach 30cm = 0.8s"
    • • Standardized Work: "Straight stitch @ 2500 RPM = 2.1s/15cm"
    • • Fundamental Movements: 18 basic motions with predefined times
  4. 04
    Intelligent Inference: AI fills gaps between frames using ergonomic models and industry knowledge (e.g., typical machine speeds, operator fatigue patterns, micro-adjustments).

Example Analysis

VISUAL INPUT:

  • Frame 1: Hand 40cm from fabric
  • Frame 2: Hand grasping fabric
  • Frame 3: Fabric under presser foot
  • Frame 4: Needle stitching
  • Frame 5: Operator removing piece
  • Frame 6: Finished piece on table

AI INFERENCE:

  • Reach 40cm → 1.2s
  • Grasp thin fabric → 0.4s
  • Position under foot → 1.5s
  • Stitch 15cm @ 2500 RPM → 2.1s
  • Remove + Release → 0.9s
  • TOTAL: 6.1 seconds

Accuracy & Limitations

PRECISION:

  • ✓ Typical variance: ±10-15%
  • ✓ Sufficient for: Inefficiency detection, operator comparison, layout optimization
  • ✓ Best for: Repetitive operations with clear phases

NOT SUITABLE FOR:

  • ✗ Legal time-study documentation (requires continuous recording)
  • ✗ Micro-motion analysis (<0.1s precision needed)
  • ✗ Highly variable operations (custom assembly, R&D)
