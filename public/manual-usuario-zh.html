1. 执行仪表板

工厂的指挥中心。实时可视化生产脉搏。

实时模拟

您看到的数字（OEE、总产量、周期）不是静态的。它们直接连接到您的平衡和成本模块。如果生产线出现瓶颈，仪表板中的周期将上升，产量将下降。

OEE (整体设备效率)

衡量制造效率。结合可用性 x 性能 x 质量。

总产量

生产件数计数。此计数器根据平衡线的实际"周期时间"推进。

执行 PDF 报告

一键生成管理层的高级摘要。报告包括 KPI、OEE 趋势和预测风险。

2. AI 视频分析

将标准视频转换为精确的工业工程数据。

  1. 上传视频： 上传手动操作（如组装、缝纫）的 MP4 视频。
  2. Gemini 分析： 我们的 AI 逐帧处理以识别基本操作动作。
  3. 结果： 接收包含标准时间、浪费检测和人体工程学建议的报告。

专业提示

确保操作员全身可见，以获得最佳人体工程学结果。

3. 生产线平衡

通过平衡工位间的工作量来优化生产流程。

直观的拖放操作

将操作（色块）从工作台拖到工位 1-4。系统自动重新计算饱和度。

  • 瓶颈： 具有最高条形的工位定义了整条线的速度。
  • 效率： 旨在使所有条形尽可能平齐。

4. 成本模拟

即时计算运营决策的财务影响。

关键输入

  • 时薪 每分钟成本的直接影响
  • 总体效率 实际降级系数
  • 间接费用 % 行政/制造成本

"单件成本"

这是您必须监控的最终数字。每次修改输入时都会更新。

5. 移动端安装 (PWA)

Manufactura IA Pro 旨在作为原生应用程序在您的移动设备上运行。无需从应用商店下载。

iPhone & iPad

  1. 打开 Safari 并输入应用网址。
  2. 点击 分享 按钮（带向上箭头的方块）。
  3. 滑动并选择 "添加到主屏幕"。
  4. 点击右上角的 添加。

Android (Chrome)

  1. 打开 Chrome 并进入应用。
  2. 点击三个垂直 点 (菜单)。
  3. 选择 "安装应用"。
  4. 在弹出的窗口中确认安装。

6. 自定义数据库

使 Manufactura IA Pro 适应您的实际情况。您不受限于我们的标准；您可以创建自己的制造宇宙。

如何添加产品和操作

  1. 转到侧边栏菜单中的 "Settings" (设置)。
  2. 查找 "Data Management" (数据管理) 部分。
  3. 要添加 新产品:
    • 输入 SKU 或名称。
    • 定义目标周期时间 (Target Cycle Time)。
  4. 定义 自定义操作:
    • 将操作（切割、组装、包装）关联到您的产品。
    • 建立每个操作的标准成本。

7. 语音命令

无需触摸屏幕即可控制界面。非常适合戴手套或双手忙碌的操作员。

激活

系统持续监听。只需大声说出命令。 您会在顶部栏看到 "Listening..." 指示器。

关键词

  • "Dashboard" / "Home" 转到仪表板
  • "Balancing" / "Line" 转到平衡

8. 工厂模式 (Factory Floor)

高对比度界面，专为远距离观看或工业照明条件下的屏幕设计。

特点

  • 高对比度： 黄/黑配色方案，提供最大可读性。
  • 纯数据： 消除审美干扰，100% 专注于指标。
  • 远距离可见： 字体放大，适合远距离阅读。

如何激活

查找开关 "Factory Floor Mode" 在：
Settings > View Settings
